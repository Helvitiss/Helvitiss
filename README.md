# Александр Корабель

Python Backend Developer. Проектирую и развиваю backend-системы и Telegram-ботов для реальных рабочих процессов: от архитектуры и API до деплоя, мониторинга и поддержки.

## Ключевая экспертиза

- Архитектура backend-сервисов: layered/clean-подход, модульность, масштабируемость.
- Разработка API и бизнес-логики: FastAPI, Django, SQLAlchemy, Pydantic.
- Асинхронные системы и боты: aiogram 3, планировщики задач, интеграции.
- Работа с данными: PostgreSQL, MySQL, SQL, оптимизация запросов.
- Инфраструктура и эксплуатация: Docker, Docker Compose, Linux, systemd.
- Качество и надежность: pytest, автотесты, предсказуемые релизы.
