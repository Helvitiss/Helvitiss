# Александр Корабель

**Python Backend Developer**

Разрабатываю production-решения на Python: backend-сервисы, Telegram-боты и инструменты автоматизации.
Фокус — на надёжной архитектуре, предсказуемой поддержке и практической пользе для бизнеса.

---

## Ключевая экспертиза

- Проектирование и развитие backend-приложений на **FastAPI** и **Django**.
- Разработка и сопровождение Telegram-ботов на **aiogram 3**.
- Построение асинхронных сервисов (**asyncio**, **SQLAlchemy 2.0**, **asyncpg**, **aiomysql**).
- Работа с реляционными БД: **PostgreSQL**, **MySQL**, **SQL**.
- Инфраструктура и деплой: **Docker**, **Docker Compose**, **Linux/systemd**.
- Качество и стабильность: **pytest**, модульное/интеграционное тестирование.
