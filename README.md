# Александр Корабель

**Python Backend Developer** с практическим опытом разработки и поддержки production-сервисов: Telegram-боты, API, автоматизация и парсинг.

Проектирую backend-решения с фокусом на надёжность, масштабируемость и понятную архитектуру. Работаю с асинхронным стеком, SQL-базами данных, контейнеризацией и тестированием.

---

## Профессиональные навыки

- Разработка backend-приложений на **Python** (FastAPI, Django, aiogram)
- Проектирование API и бизнес-логики
- Асинхронное программирование (**asyncio**, asyncpg, aiomysql)
- Интеграции с Telegram API и внешними сервисами
- Работа с БД: **PostgreSQL**, **MySQL**, SQLAlchemy 2.0
- Контейнеризация и деплой: **Docker**, Docker Compose, systemd
- Тестирование: **pytest**, покрытие критичных сценариев
